Smart Cistern App

The Smart Cistern App is an MVP that helps users to remotely drain their cisterns, manage automated drainages and check out stats about the collection of stormwater in their properties.

Smart Cistern App

DISCLAIMER: Due to an NDA, only a summary of the research is being showcased here. The client name and app brand logo were also omitted.

My Role

UX Researcher and Designer

Goal

To employ an user-centred approach in the redesign of the Smart Cistern app, based on User Research data and a Heuristic Evaluation of the existing design.

Type & Duration

Client Work - 2 months

Existing Design & Challenge

The existing design featured two main screens: the first one is an overview of the user's cistern and the second one three charts. There was a hidden menu where the user is able to select different addresses and navigate to secondary screens (Q&A and Options).

Our challenge was to make the app, whose target audience is quite broad, easier to use retaining only the features that are meaningful for the average user's experience.

Heuristic Evaluation

The first step towards evaluating the current design was to execute a Heuristic Evaluation of the existing UI. For that we used Nielsen's 10 Usability Heuristics for User Interface Design and, where applicable, Xerox's Heuristic Evaluation Checklist.

Interviews & Task Analysis

We did 3 interviews with potential and current users to try and identify:

  • What are their main goals when using the app.
  • What they currently like about the app.
  • What issues they currently face when using the app.
  • Missed opportunities that we could explore with the redesign.

Animations

The animated transitions and UI feedback play a vital role in this prototype. They aim to improve the visibility of how the user's actions effect their physical cistern (e.g. how much water will be drained vs how much water will be left).

Besides, they help users better locate themselves within the app architecture, in addition to improving the overall experience through delightful micro-interactions.
